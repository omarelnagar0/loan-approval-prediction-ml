# Loan Approval Prediction using Machine Learning

## Project Overview
This project builds a machine learning model to predict whether a loan application will be approved based on applicant financial and credit information.

The goal is to simulate how financial institutions automate credit decision-making.

## Dataset
Loan Approval Prediction Dataset (Kaggle)

Features include:
- Income
- Loan Amount
- Loan Term
- CIBIL Score
- Employment status
- Credit history

## Technologies Used
- Python
- Pandas
- Matplotlib & Seaborn
- Scikit-learn

## Machine Learning Models
Two models were trained and evaluated:

1. Logistic Regression
2. Decision Tree Classifier

The Decision Tree achieved **97% accuracy** on the test dataset.

## Key Insights
Feature importance analysis showed that the most influential variables were:
- CIBIL Score
- Loan Term
- Loan Amount
- Annual Income

This aligns with real-world banking practices where credit score is the primary indicator of loan approval.

## Business Value
Machine learning models like this can help financial institutions:
- Automate loan screening
- Reduce manual decision time
- Improve risk assessment
